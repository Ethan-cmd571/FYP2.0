ML CityBuilder - Curriculum Structure Version

This regenerated package is based on the earlier curriculum structure:
- Foundations / Easy
- Intermediate / Medium
- Advanced / Hard

Included files:
- index.php
- app.js
- style.css
- api.php
- config.php
- ml_citybuilder_curriculum_structure.sql
- README.txt

Setup:
1. Copy the PHP, JS, and CSS files into your XAMPP htdocs project folder.
2. Import ml_citybuilder_curriculum_structure.sql into phpMyAdmin.
3. Ensure Apache and MySQL are running.
4. Open http://localhost/your-folder/index.php

Features:
- Curriculum-based progression by difficulty
- 150 structured ML tasks with lesson content
- Lesson -> Question -> Explanation flow
- 30-second lesson timer
- Topic mastery dashboard
- Revenue, happiness, and XP rewards / penalties
- District skyline growth

- Leaderboard
- Accuracy statistics panel
- Activity stream

- Badges panel
- Course completion badges for Foundations, Intermediate, and Advanced
